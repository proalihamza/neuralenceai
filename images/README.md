# Images Required for Neuralence Landing Page

Please place the following images in this folder:

## Logo
- **logo.png** - Your company logo (recommended size: 200x80px or similar aspect ratio)

## Service Images
- **dotnet-mvc.png** - .NET MVC technology image
- **dotnet-core.png** - .NET Core technology image  
- **angular.png** - Angular framework image
- **mean-stack.png** - MEAN stack image
- **mern-stack.png** - MERN stack image
- **ai-ml.png** - AI/ML technology image

## Hero Section (Optional)
- **hero-image.png** - Main hero section image (recommended size: 800x600px)
- **hero-bg.jpg** - Background image for hero section

## Recommended Image Specifications
- Format: PNG or JPG
- Service images: 200x200px or similar square format
- Logo: Transparent PNG preferred
- All images should be optimized for web (under 500KB each)

## Image Sources
You can find suitable images from:
- Official technology logos and icons
- Stock photo websites (Unsplash, Pexels)
- Icon libraries (Flaticon, Icons8)
- Custom graphics created for your brand
